## Current Purpose

This repository (`working_papers`) now functions exclusively as the experimental staging ground and research sandbox for the SDI framework. 

It contains raw conceptual data, unverified hypotheses, and the developmental baseline for **Working Paper Six** (Financial Modeling and Capitation Rate Analysis).

Do not use the contents of this repository for municipal planning, pilot deployment, or structural evaluation. All authoritative architecture must be pulled from the `sdi-specification` repository.
