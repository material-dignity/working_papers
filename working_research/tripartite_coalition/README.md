# Working Paper 7: Tripartite Coalition Architecture

This directory houses the legal, financial, and operational research for the deployment of the Tripartite Coalition model.

## Directory Structure
*   `/research/`: Contains foundational metrics, jurisdictional analysis, and financial modeling for the prototype deployment.
*   `/drafts/`: (Future) Operational agreements, pro forma sheets, and working paper manuscripts.

## Core Directives
1.  **Legal Structure**: Define incorporation jurisdiction and charter requirements for the Public Benefit Corporation and the independent Stewardship Trust.
2.  **Financial Underwriting**: Model the targeted 8.0% IRR, including the required debt-to-equity ratio for distressed asset acquisition.
3.  **Clinical Metrics**: Establish strict metabolic and clinical acuity thresholds for activating the Recuperative Care CalAIM pathway.
