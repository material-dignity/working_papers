# Field Manual 2: Social Network Transition Protocol (SNA Pipeline)

## Operational Baseline
Housing lotteries destroy relational infrastructure by randomizing assignment. The SNA Assessment Pipeline maps existing encampment networks to preserve them. The intake worker does not evaluate personality. The intake worker maps dependency.

This matrix dictates the mandatory interview script, the data verification protocol, and the Process 4 assignment routing constraints.

---

## 1. The Intake Mapping Script (Data Collection)
Questions must map material reliance, not subjective friendship.

| Domain | Mandatory Script | Target Output |
| :--- | :--- | :--- |
| **Material Security** | "When you leave your tent, who watches your gear?" | Identifies primary trust node. |
| **Metabolic Supply** | "Who do you share food or tobacco with?" | Identifies routine reciprocal nodes. |
| **Crisis Response** | "If you get sick or hurt, who do you find first?" | Identifies apex reliance node. |
| **Toxicity Identification** | "Is there anyone here you need to stay away from?" | Identifies predator/parasite nodes requiring isolation. |

---

## 2. The Verification and Scoring Matrix
Data requires bidirectional verification. Unverified claims are algorithmically discarded.

| Metric | Action | Algorithmic Weight |
| :--- | :--- | :--- |
| **Reciprocity (Verified)** | Worker asks Node A if they watch Node B's gear. Node A confirms. | **+3** (Primary Cluster Anchor) |
| **Reciprocity (Unverified)** | Worker asks Node A. Node A denies or is unavailable. | **0** (Discard Tie) |
| **Density (Volume)** | Node B is named by three or more individuals in the encampment. | **+2** (Central Hub Node) |
| **Toxicity (Risk)** | Node C is named as a threat by any verified node. | **-5** (Isolation Flag) |

---

## 3. Process 4 Assignment Routing (The Algorithm)
Intake workers input the scored data into the Process 4 cybernetic grid. The algorithm executes assignment based on the following rigid hierarchy:

1. **Priority One (The Isolation Constraint):** Toxicity flags trigger absolute geographic separation. Node C is assigned to a different floor or different building than the naming nodes. **Overrides all other rules.**
2. **Priority Two (The Anchor Constraint):** Mutually verified Reciprocity (+3) pairs are mandatorily assigned to the same 12-person Dunbar pod. 
3. **Priority Three (The Hub Constraint):** High-Density nodes (+2) are assigned to pods requiring stabilization, deploying their existing social gravity to anchor isolated individuals.
4. **Priority Four (The Remnant Constraint):** Individuals with zero verified ties are distributed evenly across high-density pods. No pod may consist entirely of isolated individuals.

---

## 4. Execution Failure Rule
Assigning mutually verified bond partners to separate floors constitutes an architectural failure. The relational layer is the primary driver of the 24-month retention metric. Destroying the layer at intake guarantees retention failure.
